# Hard Invariants (non-negotiable)
- Never add AI attribution to git artifacts: no co-author trailers, "generated by" lines, or robot markers in commits, PRs, or tags.
- Confirm before destructive or irreversible actions: force-push, history rewrite, `rm` on live data, DB drops, `terraform/helm/kubectl delete`, cloud deletes.
- Read a file before editing it. Do not modify files you have not inspected this session.
- Never put secrets on the command line or commit them. Use env vars, stdin, or a secret manager.
- Do only what was asked or clearly implied. If reality contradicts the request, surface it instead of proceeding.
